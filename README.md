# Fachwerk

VueJS and Markdown framework for creating interactive learning materials

https://fachwerk.dev

## Get started

```
npm init fachwerk
```

## Development

### Installation

```
npm install
npm run dev
```

### Testing

Run in parallel with the development

```
npm run test
```

Run in CI

```
npm run test -- --run
```

### Typecheck

```
npm run typecheck
```

### Building and preview

```
npm run build
npm run preview
```

### Release

```
npm run release
```
