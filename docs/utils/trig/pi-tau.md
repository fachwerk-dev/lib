# PI and TAU

## PI

Returns <f-math>\pi</f-math> value, same as `Math.PI`.

```ts
const PI: number;
```

#### Usage

```md
{{ PI }}
```

## TAU

Returns <f-math>\tau = 2 \times \pi</f-math> value. Inspired by the [Tau Manifesto](https://tauday.com/tau-manifesto).

```ts
const TAU: number;
```

#### Usage

```md
{{ TAU }}
```
