# rectgridpoints

`rectgridpoints()` outputs an array of `Point`'s in 2D grid.

```ts
type Point = [x: number, y: number];

function rectgridpoints(
  length: number,
  step: number,
  position: Point = [0, 0]
): Point[];
```

## Usage

```md
{{ rectgridpoints(2,10) }}
```

## Example

Using `rectgridpoints()` with `10 Ã— 10` grid with step `20` and `10,10` offset from top left corner:

```md
<svg height="200">
  <circle
    v-for="[x,y] in rectgridpoints(10,20,[10,10])"
    :cx="x"
    :cy="y"
    r="10"
    fill="lightblue"
  />
</svg>
```
